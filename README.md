# Portfolio

Portfólio pessoal desenvolvido com foco em design minimalista e experiência do usuário fluida.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Características

- Design minimalista e moderno
- Animações de scroll interativas
- Scroll snap no mobile para navegação fluida
- Menu hamburguer responsivo
- Totalmente responsivo (desktop e mobile)
- Navegação horizontal no desktop
- Navegação vertical com snap no mobile

## Responsividade

O portfólio adapta-se automaticamente para diferentes tamanhos de tela:
- **Desktop**: Navegação horizontal com animações de largura
- **Mobile**: Navegação vertical com scroll snap automático

## Seções

- **Home**: Apresentação inicial
- **Perfil**: Sobre mim e hard skills
- **Projetos**: Destaque do projeto Artsync
- **GitHub**: Projetos acadêmicos e pessoais
- **Portfólio**: Informações sobre este projeto
- **Contato**: Links para email, LinkedIn e GitHub
