PROJETOS

Artsync

Feito como projeto de fim de ano do Colegio Cotemig, o Artsync é uma plataforma web/rede social gratuita pensada para artistas independentes, onde é possível gerenciar portfólios, organizar eventos, trocar conhecimentos com outros criativos e receber insights personalizados de carreira por meio de inteligência artificial. O projeto foi desenvolvido com arquitetura MVC, utilizando PHP, MySQL e CSS, além da aplicação do padrão de projeto Singleton para garantir melhor organização e controle das instâncias.
